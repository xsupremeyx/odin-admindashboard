# odin-admindashboard
A admin dashboard layout built using **HTML** and **CSS Grid/Flexbox**.  
The project focuses on clean grid layout structure.

## Features
- CSS Grid–based page layout
- Flexbox for internal component alignment
- Fixed header and sidebar
- Scrollable main content section
- Card-based project layout
- Announcement and trending sections
- SVG icon support
- Consistent spacing, shadows, and border radius

## Tech Stack
- HTML5
- CSS3 (Grid & Flexbox)
- Google Fonts (Roboto)

## Folder Structure
project/
├── index.html
├── styles.css
└── assets/
└── icons/
├── *.svg

## Notes
- SVG icons are used via `<img>` tags
- Icons support color theming using `currentColor`
